# [level 1] 약수의 합 -12928

### 문제설명

정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

### 제한 사항

- n은 0이상 3000이하인 정수입니다.

### 입출력 예

| n   | return |
| --- | ------ |
| 12  | 28     |
| 5   | 6      |

입출력 예 #1
12의 약수는 1, 2, 3, 4, 6, 12입니다. 이들을 모두 더하면 28이 됩니다.

입출력 예 #2
5의 약수는 1, 5입니다. 이들을 모두 더하면 6이 됩니다.
